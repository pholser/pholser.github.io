--- layout: post title: JOpt Simple 2.4 released published: 1 category: - Programming ---

JOpt Simple is my Java library for parsing command-line options.  Release 2.4 sports some bug fixes, internal reorganization, some new features such as option synonymy and a cleaned-up help printing facility.  JOpt Simple seems to be generating more interest over time -- I'm glad it's helping programmers out with an uncomplicated API and more-than-adequate feature set.
